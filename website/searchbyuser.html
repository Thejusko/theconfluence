Filter instructions and notes

You can press enter to autocomplete when there is only one possible user for the given filters.
The random button only selects people that satisfy all the filters.
On the bottom right the names of people that satisfy all the filters, including name, appear.

Use the "minimum stay" filter to remove from possible results people that have not stayed long enough in the sub.
Use the "maximum lowest flair" filter to remove from possible results people that have never reached a flair number lower than the one selected.
You can manually type the name of someone even if they don't satisfy the filters.

The source data for this graph can be found here. (Might take a while to load.)
The exact flair numbers of the user on screen appear in the console. (F12 > "Console"). 0 means not part of the sub.
